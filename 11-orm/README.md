## Main Topic:
* Mongo -- a *noSQL* database
* Stores "documents" instead of records/rows
* Pros
  * Complete Data Sets
  * No Joins
  * Logical Modeling
  * Variable Modeling
* Cons
  * Not easily indexable
  * Volatile shape/schema
